# EMAILS-SCRAPING
USE THIS LEGITIMATELY. This is intended onlyfor legitimate use. 

This scrapes emails from any country at any amount needed.

**USAGE**
  This is a python file. it has no external dependency. Just install python. Python3 was use for this case

**RUN**
chmod +x ./email_fetcher.py
./email_fetcher.py [country] [numbers]

**EXAMPLES**

./email_fetcher.py us 50000
./email_fetcher.py za 10000
./email_fetcher.py uk 20000
